# Error-Mode Taxonomy (E1–E6)

A six-category taxonomy was developed to classify the underlying cause of incorrect model responses, rather than relying on a simple correct/incorrect judgement.

| Code | Category | Description |
|---|---|---|
| E1 | Conceptual Error | Wrong concept, wrong theorem, logic error |
| E2 | Method Error | Wrong method, wrong substitution, wrong procedure, incorrect derivative/integration approach |
| E3 | Calculation Error | Calculation mistake, arithmetic error, wrong numerical values, reciprocal error |
| E4 | Reasoning Error | Incorrect explanation, different explanation leading to wrong reasoning, weak justification |
| E5 | Incomplete Solution | Incomplete answer, unfinished proof, only a theory without a final solution |
| E6 | Hallucination | Fabricated information, unsupported assumptions — model invented a non-existent formula or theorem, or produced content completely unrelated to the question |
